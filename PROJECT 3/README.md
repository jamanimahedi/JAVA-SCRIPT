# 🌤 Weather Check App

A simple and modern weather application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the OpenWeather API.

---

live link = https://weather-api-project-js.netlify.app/

## 🚀 Features

* 🌍 Search weather by city name
* 🌡 Displays temperature in Celsius
* 🌤 Dynamic weather icons (cloud, rain, clear)
* 💧 Shows humidity
* 🌬 Displays wind speed
* ❌ Error handling for invalid city names
* 🎨 Modern glassmorphism UI design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Glass effect UI)
* JavaScript (Async/Await, Fetch API)
* OpenWeather API

---

## 📸 UI Preview

* Clean and centered layout
* Background image with blur effect
* Responsive input + button design

---

## ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeather API
3. Data is fetched using `fetch()`
4. Weather details are displayed dynamically

---

## 🔑 API Setup

This project uses the **OpenWeather API**.

👉 Replace your API key in `script.js`:

```js
appid=YOUR_API_KEY
```

Get your API key from: https://openweathermap.org/api
